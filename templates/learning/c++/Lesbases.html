I. Introduction

Le langage C a été créé par Dennis Ritchie au début des années 1970. Il est largement utilisé dans le développement logiciel en raison de sa simplicité, de sa flexibilité et de sa performance. Il a également été le point de départ de nombreux autres langages de programmation.

II. Structure d'un Programme C

Un programme C est généralement structuré en plusieurs parties :

// Inclusion de bibliothèques
#include <stdio.h>

// Déclaration des fonctions (prototypes)
int additionner(int a, int b);

// Fonction principale
int main() {
    // Instructions du programme
    int resultat = additionner(5, 3);
    printf("Le résultat est : %d\n", resultat);

    return 0; // Fin du programme
}

// Définition de la fonction additionner
int additionner(int a, int b) {
    return a + b;
}

III. Variables et Types de Données

En C, vous devez déclarer explicitement le type de chaque variable :

int entier = 10;
float reel = 3.14;
char caractere = 'A';

IV. Opérateurs

Les opérateurs permettent d'effectuer des opérations sur les variables :

int somme = entier + 5;
int produit = entier * 2;

V. Structures de Contrôle

1. Les Boucles

for (int i = 0; i < 5; i++) {
    // Instructions à répéter
}

while (condition) {
    // Instructions tant que la condition est vraie
}

do {
    // Instructions exécutées au moins une fois
} while (condition);

2. Les Conditions

if (condition) {
    // Instructions si la condition est vraie
} else if (autre_condition) {
    // Instructions si l'autre condition est vraie
} else {
    // Instructions sinon
}

VII. Fonctions

Les fonctions permettent de structurer votre code et de le rendre plus lisible et réutilisable.

VIII. Structures

Les structures permettent de regrouper différentes variables sous un même nom.

IX. Pointeurs

Les pointeurs sont des variables qui contiennent l'adresse mémoire d'une autre variable.

int nombre = 10;
int *pointeur = &nombre;

X. Allocation Dynamique de Mémoire

L'allocation dynamique de mémoire en C permet au programme de demander et de libérer de la mémoire à la volée, offrant une flexibilité essentielle pour gérer efficacement les données pendant l'exécution du programme.

int *pointeur_entier = (int *)malloc(sizeof(int));
