Type variables !

Les Différents type de variables

1. Types de Données Fondamentaux

a. Entiers

  • int : Entier signé (4 octets)
  • short : Entier court (2 octets)
  • long : Entier long (4 octets)
  • long long : Entier long long (8 octets)

b. Floats

  • float : Nombre à virgule flottante simple précision (4 octets)
  • double : Nombre à virgule flottante double précision (8 octets)
  • long double : Nombre à virgule flottante longue double précision (Variable, généralement 10 ou 12 octets)

c. Caractères

  • char : Caractère (1 octet)

2. Types de Données Composés

a. Tableaux

  • int tableau[5] : Tableau d'entiers de taille 5 (5 * 4 octets (pour un int))

b. Chaînes de Caractères

  • char chaine[] = "Hello" : Chaîne de caractères (Variable, dépend de la longueur de la chaîne)

3. Types de Données Pointeurs

  • int *pointeur : Pointeur vers un entier (4 octets (sur une architecture 32 bits) ou 8 octets (sur une architecture 64 bits))
  • char *pointeur_chaine : Pointeur vers une chaîne de caractères (4 octets (sur une architecture 32 bits) ou 8 octets (sur une architecture 64 bits))

4. Types de Données Void

  • void : Utilisé dans les fonctions qui ne retournent rien (Généralement utilisé pour indiquer qu'une fonction ne retourne rien)

Exemples d'Utilisation

a. Entiers

int entier = 42;

b. Floats

float nombre_flottant = 3.14;

c. Caractères

char caractere = 'A';

d. Tableaux

int tableau[5] = {1, 2, 3, 4, 5};

e. Chaînes de Caractères

char chaine[] = "Hello";

f. Pointeurs

int nombre = 10;
int *pointeur = &nombre;
