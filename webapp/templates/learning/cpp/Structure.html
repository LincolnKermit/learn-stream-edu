Structure et type de fichier !

I. Introduction

Les structures en langage C sont des éléments fondamentaux permettant de regrouper des variables de types différents sous un même nom. Elles offrent une manière de créer des entités plus complexes, facilitant ainsi la manipulation de données dans un programme.

II. Déclaration de Structures

Pour déclarer une structure, on utilise le mot-clé struct suivi du nom de la structure et d'un ensemble de variables appelées membres.

struct Point {
    int x;
    int y;
};

III. Déclaration et Initialisation des Structures

// Déclaration
struct Point p1;

// Initialisation
p1.x = 10;
p1.y = 20;

// Déclaration et initialisation simultanées
struct Point p2 = {5, 15};

IV. Accès aux Membres d'une Structure

// Accès aux membres
int coordX = p1.x;
int coordY = p1.y;

V. Structures Anonymes

Il est possible de déclarer une structure sans spécifier son nom, créant ainsi une structure anonyme.

struct {
    char nom[50];
    int age;
} personne;

VI. Structures Nommées et Utilisation

// Déclaration d'une structure nommée
struct Etudiant {
    char nom[50];
    int age;
    float moyenne;
};

// Utilisation de la structure
struct Etudiant etudiant1;
strcpy(etudiant1.nom, "Alice");
etudiant1.age = 20;
etudiant1.moyenne = 16.5;

VII. Structures Imbriquées

Les structures peuvent être imbriquées, permettant de créer des structures plus complexes.

struct Date {
    int jour;
    int mois;
    int annee;
};

struct Etudiant {
    char nom[50];
    int age;
    float moyenne;
    struct Date dateNaissance;
};

VIII. Utilisation de Pointeurs avec les Structures

Les pointeurs peuvent être utilisés pour manipuler des structures.

struct Point {
    int x;
    int y;
};

struct Point p = {3, 7};
struct Point *ptrPoint = &p;

// Accès aux membres à travers un pointeur
int coordX = ptrPoint->x;
int coordY = ptrPoint->y;

IX. Taille d'une Structure

La taille d'une structure est la somme des tailles de ses membres, avec un alignement possible pour des raisons d'efficacité.

X. Utilisation de Structures dans les Tableaux

Les structures peuvent être utilisées pour créer des tableaux de types structurés.

struct Personne {
    char nom[50];
    int age;
};

struct Personne groupe[5];

XI. Avantages des Structures

  • Regroupement logique : Permet de regrouper des données liées.
  • Modularité : Facilite la création de programmes modulaires.
  • Clarté du Code : Améliore la lisibilité du code.

On peut utiliser la syntaxe typdef aussi !

Cette syntax typedef est utiliser pour créé un type de variable on peut donc l’utiliser pour les structure.

La syntaxe générale de typedef est la suivante :

typedef ancien_type nouveau_nom;

Voici quelques exemples pour illustrer son utilisation :

1. Utilisation avec des Structures

typedef struct {
    int jour;
    int mois;
    int annee;
} Date;

// Maintenant, on peut déclarer une variable de type Date de la manière suivante :
Date anniversaire;

2. Utilisation avec des Tableaux

typedef int TableauEntiers[5];

// Maintenant, on peut déclarer un tableau d'entiers de la manière suivante :
TableauEntiers mesNombres = {1, 2, 3, 4, 5};

3. Utilisation avec des Pointeurs

typedef int* PointeurEntier;

// Maintenant, on peut déclarer un pointeur vers un entier de la manière suivante :
PointeurEntier monPointeur;

4. Utilisation avec des Fonctions

typedef int (*FonctionEntiere)(int, int);

// Maintenant, on peut déclarer une fonction prenant deux entiers et retournant un entier de la manière suivante :
FonctionEntiere addition = somme;

Structure dans un header

Il existe des fichier avec extension .h ou l’on peut enregistrer nos structure et faire en sorte de les definir si elle ne le sons pas.

fichier —> struct.h

#ifndef STRUCT_H
#define STRUCT_H

// Le contenu de votre fichier d'en-tête (déclarations de structures, fonctions, etc.) serait placé ici.

#endif // STRUCT_H

main —> main.c

#include "struct.h"

int main(){
}

Les fonctions aussi

fichier —> fonction.c

int factorielle(int n){
	if(n == 0 || n == 1) return 1;
	
	return n * factorielle( n - 1);
}

header ffonction —> fonction.h

#include "fonction.c"

int factorielle(int n);

fichier main —> main.c

#include "fonction.h"
