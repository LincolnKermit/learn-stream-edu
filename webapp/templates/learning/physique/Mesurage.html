Type A - Méthode Statistique

  1. Premièrement, entrer les données dans la calculatrice ou un logiciel d'analyse statistique. Cela vous permettra de traiter un ensemble de mesures.
  2. Calculer l'écart-type de l'échantillon en utilisant la fonction dédiée dans la calculatrice ou le logiciel. L'écart-type mesure la dispersion des valeurs autour de la moyenne.
  3. Le format d'écriture des résultats est le suivant :

    Rexp = Valeur moyenne de l'échantillon ; u(Rexp) = écart-type de l'échantillon

    Exemple : Rexp = 2.51 V ; u(Rexp) = 0.025 V

  4. Pour obtenir l'incertitude standard du **Type A**, utiliser la formule suivante :

    𝕌(R) / √ (Nombre de valeurs)

    Note : Cette formule calcule l'incertitude type pour un ensemble de données. Divisez l'écart-type par la racine carrée du nombre de valeurs pour obtenir l'incertitude standard.

  5. Calcul du Z-score :
    • Formule du Z-score : Z = (X - µ) / 𝕌(R)
    • Explication : Le Z-score mesure combien d'écarts-types une valeur X est éloignée de la moyenne µ. Si Z est proche de 0, la valeur est proche de la moyenne.
  6. Comparaison entre deux valeurs mesurées :

    Δ = |R1 - R2| / √[u(R1)² + u(R2)²]

    Explication : Cette formule compare la différence entre deux mesures en tenant compte de leurs incertitudes respectives. Si Δ <= 2, les valeurs sont considérées comme compatibles.

Type B - Méthode Déterministe

  1. Le **Type B** est utilisé lorsqu'il n'y a qu'une seule valeur de mesure ou des informations provenant de sources externes (par exemple, manuels, données fournies). Dans ce cas, l'incertitude est souvent liée à la précision de l'instrument.
  2. Estimer l'incertitude en fonction de la précision de l'instrument ou des spécifications fournies. Vérifiez les données techniques de l'appareil pour obtenir ces informations.
  3. Le format d'écriture des résultats est le suivant :

    Rexp = Valeur mesurée ou estimée; u(Rexp) = Incertitude liée à l'instrumentation ou aux données externes

    Exemple : Rexp = 5.00 V ; u(Rexp) = 0.1 V

  4. La formule d'incertitude peut être similaire au **Type A** si les informations statistiques sont disponibles :

    𝕌(R) / √ (Nombre de valeurs)

    Note : Utilisez cette formule uniquement si vous disposez de plusieurs mesures statistiques pour le Type B.

  5. Comparaison entre deux valeurs mesurées (Type A et B) :

    Δ = |R1 - R2| / √[u(R1)² + u(R2)²]

    Explication : Utilisez cette formule pour comparer une mesure issue du Type A à une du Type B. Cela vous permet de vérifier si les deux valeurs sont compatibles en prenant en compte leurs incertitudes.

Formule commune

𝕌(R) / √ (Nombre de valeurs)

Cette formule est utilisée pour calculer l'incertitude des deux types lorsqu'il y a plusieurs valeurs disponibles.
